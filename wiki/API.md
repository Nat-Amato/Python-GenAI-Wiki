Termine: API (Application Programming Interface)

Descrizione:
Un'API è un insieme di regole e protocolli che permette a due applicazioni software diverse di comunicare tra loro e scambiarsi dati in modo sicuro e standardizzato. Funziona come un intermediario, simile a un cameriere in un ristorante: riceve la tua richiesta, la porta al sistema di destinazione (la cucina) e ti restituisce il risultato, nascondendo all'utente tutta la complessità del processo informatico sottostante.

Contesto:
Sviluppo software, integrazione di sistemi, sviluppo web e mobile, servizi cloud.

Esempio:
Quando prenoti un volo tramite un sito di comparazione (come Skyscanner o Booking), quel sito utilizza le API delle singole compagnie aeree per interrogare i loro database in tempo reale, permettendoti di vedere tutte le tariffe disponibili in un'unica pagina.
